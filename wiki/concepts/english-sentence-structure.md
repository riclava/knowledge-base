English sentence structure 是英语句子成立和扩展的骨架，核心是限定谓语、主语、句型成分、短语修饰和从句嵌套之间的关系。

## Definition

来源把完整英语句子的最小核心写成：

```text
Subject + Predicate
```

其中谓语必须包含限定动词。学习复杂句时，先确认这一层是否清楚，再处理修饰语和从句。

## Five Basic Sentence Patterns

| 句型 | 结构 | 说明 |
|---|---|---|
| SV | 主语 + 不及物动词 | 动作不需要宾语 |
| SVC | 主语 + 系动词 + 表语 | 说明主语是什么或怎么样 |
| SVO | 主语 + 及物动词 + 宾语 | 动作作用于对象 |
| SVOO | 主语 + 动词 + 间宾 + 直宾 | 给某人某物 |
| SVOC | 主语 + 动词 + 宾语 + 宾补 | 补充说明宾语状态或身份 |

复杂句通常也能还原到这五类骨架，再叠加定语、状语、同位语和从句。

## Constituents and Units

| 单位 | 特点 | 文档价值 |
|---|---|---|
| word | 单个词 | 判断词类 |
| phrase | 没有完整主谓结构的词组 | 承担主语、宾语、定语、状语等功能 |
| clause | 有主谓结构 | 可独立，也可嵌入大句 |
| sentence | 能独立表达完整意思 | 写作和标点检查的基本单位 |

## Sentence-Type Coverage

来源同时按结构和功能讲句子：

- 结构：简单句、并列句、复合句、并列复合句。
- 功能：陈述句、疑问句、祈使句、感叹句。
- 变体：否定句、一般疑问句、特殊疑问句、选择疑问句、反意疑问句、间接疑问句。

## Why It Matters

- 主语和谓语不清，会导致流水句、逗号拼接、主谓不一致和从句残缺。
- 句型意识能帮助学习者判断动词后面需要宾语、表语、宾补还是不需要补足成分。
- 对写作而言，句子结构是清晰度的第一层；篇章衔接建立在句子成立之上。

## Documentation Implications

- 讲句子结构时应同时写“英文术语”和“中文功能”，例如 `subject / 主语`、`object complement / 宾补`。
- 改错材料应要求学习者先标出限定谓语和主语核心，再讨论更细的用法。
- 对长句，应明确区分主句核心、修饰成分和从句边界。

## Common Misconceptions

| 误解 | 更准确的理解 |
|---|---|
| 词类和句子成分是一回事 | 词类是词本身的类型，句子成分是它在句子中的工作 |
| 长主语后面的近处名词决定谓语单复数 | 主谓一致看主语核心，不看介词短语里的近处名词 |
| 连接副词可以用逗号连接两个完整句 | `however`、`therefore` 等连接副词不能像 `and` / `so` 那样直接并列两个独立分句 |
