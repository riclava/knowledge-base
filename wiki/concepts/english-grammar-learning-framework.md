English grammar learning framework 是把英语语法理解为“词类 -> 句子成分 -> 结构关系 -> 意义选择”的分层系统，用来帮助学习者先看清句子结构，再做表达选择。

## Definition

在当前来源中，英语语法学习的目标不是背完术语，而是知道每个语法点解决什么表达问题。它把语法拆成四层：

| 层级 | 解决的问题 | 例子 |
|---|---|---|
| 词类 | 每个词能做什么 | noun, verb, adjective, preposition |
| 句子成分 | 词组在句子里承担什么功能 | subject, object, complement, modifier |
| 结构关系 | 信息如何连接和嵌套 | coordination, subordination, relative clause |
| 意义选择 | 说话者如何表达时间、态度和重点 | tense, aspect, voice, mood, modality, emphasis |

## Learning Sequence

来源建议的顺序是：

1. 先学五大基本句型和句子主干。
2. 再学名词、动词、形容词、副词、介词、连词等核心词类。
3. 之后进入动词时态、语态、情态和非谓语。
4. 再学习名词性从句、定语从句和状语从句。
5. 最后补倒装、强调、省略、虚拟语气、间接引语、标点、篇章衔接和正式风格。

这个顺序把“识别结构”放在“背术语”之前，适合长期复习和写作改错。

## Core Diagnostic Questions

阅读长句时，来源给出五个问题：

| 问题 | 目的 |
|---|---|
| 谓语动词在哪里？ | 找到句子是否成立的核心 |
| 主语是谁？ | 确定句子谈论对象 |
| 宾语、补语或表语是什么？ | 判断基本句型 |
| 修饰成分修饰谁？ | 避免长定语、介词短语和分词短语误导 |
| 从句从哪里开始，到哪里结束，充当什么成分？ | 判断嵌套关系和信息层级 |

## Why It Matters

- 它帮助学习者把复杂句还原成可分析的骨架。
- 它把阅读、写作、改错和复习放进同一个框架里。
- 对技术写作来说，这个框架能把“英文是否准确”拆成可检查项目，而不是只依赖语感。

## Documentation Implications

- 语法文档应优先说明规则解决的表达问题。
- 概念页应交叉链接到句子结构、动词系统、从句系统和写作风格，而不是把它们做成孤立章节。
- 练习材料应提供拆句模板、改写模板和写作自查清单，帮助学习者复用同一套分析动作。

## Common Misconceptions

| 误解 | 更准确的理解 |
|---|---|
| 语法学习就是背规则和术语 | 关键是识别结构、理解功能、做出表达选择 |
| 长句只能靠翻译理解 | 长句应先找限定谓语和主语核心，再处理修饰和从句 |
| 正式英语越复杂越好 | 优秀正式英语通常准确、清楚、克制 |
