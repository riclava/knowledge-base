English writing clarity and style 是把语法准确性、标点边界、信息组织和正式程度放在同一条写作检查链中，目标是写出清楚、克制、可读的英文。

## Rule

英文写作应优先保证句子成立、指代清晰、并列平行、标点正确和信息焦点明确；正式风格不应靠过度被动、过度名词化或堆叠长句实现。

## When to Apply

- 修改英文段落、技术说明、邮件、教程或培训材料时。
- 将中文内容改写成英文时。
- 检查学习者英文作文或技术文档英文表述时。
- 判断一个句子是否应该拆短、改为主动、补限定词或调整连接词时。

## Checklist

| 检查项 | 判断问题 |
|---|---|
| 限定谓语 | 每个完整句是否有一个清楚的限定谓语？ |
| 主谓一致 | 谓语是否和主语核心一致？ |
| 限定词 | 单数可数名词是否有冠词或其他限定词？ |
| 时态与体 | 动词形式是否符合时间、持续性和结果关系？ |
| 指代 | 代词是否有唯一、清楚的先行词？ |
| 从句 | 从句是否完整，关系词是否缺失或重复？ |
| 平行结构 | 并列项的语法形式是否一致？ |
| 标点 | 是否存在逗号拼接或流水句？ |
| 介词搭配 | 搭配是否自然，是否受中文直译影响？ |
| 清晰度 | 句子是否可以更短、更直接？ |

## Punctuation Guidance

- 用逗号连接较长并列句、前置状语从句、非限制性定语从句、插入语和列表。
- 不要用逗号直接连接两个完整句；这会形成 `comma splice`。
- 两个关系紧密的独立分句可用分号连接。
- 连接副词如 `however`、`therefore` 通常需要句号或分号提供独立分句边界。
- 引号内外标点应按美式或英式风格保持一致。

## Style Guidance

- 正式不等于复杂；优秀正式英语通常准确、清楚、克制。
- 能用主动语态清楚表达责任主体时，不要为了正式感过度使用被动。
- 能用动词直接表达动作时，不要把动作过度名词化。
- 连接词用于表达真实逻辑关系，不应为了显得“高级”而堆砌。
- 段落应保持稳定主题链，先放已知信息，再引入新信息。

## Examples

| Weak | Stronger | Reason |
|---|---|---|
| `I was tired, I went home.` | `I was tired, so I went home.` | 修复逗号拼接 |
| `The implementation of the solution was conducted by the team.` | `The team implemented the solution.` | 主动句更清楚 |
| `The test failed, therefore we revised the code.` | `The test failed; therefore, we revised the code.` | 连接副词不能只靠逗号连接两个独立分句 |
| `When the file was moved to the folder, it disappeared.` | `The file disappeared after it was moved to the folder.` | 代词指代更清楚 |
